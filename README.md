# Dashboard Stats

Dashboard amministrativa multi-sito per il monitoraggio di visite e lead.

## Configurazione
- Database: MySQL (importa `schema.sql`)
- PHP: 8.0+
- Dominio: `dashboard.bbproservice.it`

## Gestione Siti
1. Accedi come Superadmin (`admin` / `admin123`)
2. Aggiungi i siti da monitorare
3. Inserisci lo script di tracciamento nei siti remoti (istruzioni in `manage_sites.php`)
